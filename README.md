# AI-assisted Product Selection Analysis

This project analyzes product selection logic for a simulated 1688 e-commerce scenario. It focuses on identifying high-potential products based on sales volume, repurchase rate, gross margin, fulfillment efficiency, rating, return rate, and user scenario fit.

## Business Problem

How can an e-commerce team identify products that are suitable for a campus-oriented marketing campaign?

## Methodology

- Clean and structure product-level data
- Compare products across sales, repurchase rate, margin, fulfillment, rating, and return rate
- Build a weighted scoring model
- Rank products and identify top candidates
- Translate data findings into business recommendations

## Tools

Python, pandas, Excel-style business analysis, basic SQL logic, and data visualization.
